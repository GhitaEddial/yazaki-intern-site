Master Data Team Process

Complete workflow from material creation to activation

1

Material Creation (MM01)

Transaction: MM01

Views: Basic1, Basic2, Classification

Required Fields:
  • Description
  • Basic unit
  • Material group
  • Division (02 default, 03 if material group starts with 7G)
  • Weight (2g default, 10g for twisted wire, 100g for harness/coaxial)
2

Material Extension

Link material to requested plant

Note: This process can be automated

3

PIR Creation (ME11)

Transaction: ME11

Create Purchasing Info Records containing:

  • Price information
  • SPS (Supplier Packaging Specification)
  • MOQ (Minimum Order Quantity)
  • Planning details
4

SA Generation (ME31L)

Transaction: ME31L

Generate Scheduling Agreement

5

Source List Activation (ME01)

Transaction: ME01

Activate source list for procurement
